## Description

1. User uploads a resume.
2. Resume text is extracted.
3. Data is cleaned.
4. Google Gemini generates structured portfolio content.
5. Portfolio source code (HTML, CSS, JavaScript) is generated.
6. Project files are created.
7. Live preview is launched.
8. The generated portfolio can be downloaded as a ZIP.
